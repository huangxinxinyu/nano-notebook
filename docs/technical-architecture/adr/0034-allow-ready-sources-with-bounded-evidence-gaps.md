# Allow ready Sources with bounded Evidence gaps

Source verification may publish a `Ready` Source when extraction omitted only precisely identified regions and the remaining evidence is usable. The Normalized Source Artifact records Evidence Coverage, the product exposes every known gap, and Retrieval indexes only published Evidence Units; unknown coverage or loss of primary content instead makes the Source `Failed`. This preserves useful evidence from imperfect real-world documents and media without silently presenting partial extraction as complete.
