# Derive answer evidence from Run and Citations

Assistant Messages will not store an `answer_mode`. Whether an answer is source-grounded is established by its producing Agent Run, pinned Run Evidence Set, Publication Barrier, and validated Citations; the source-less product discloses its limitation at the workspace level, while Action use belongs to Checkpoints and Durable Agent Trace. Sprint 3 removes the existing Message field, API projection, and per-Message label so duplicated metadata cannot drift from authoritative execution and evidence state.
