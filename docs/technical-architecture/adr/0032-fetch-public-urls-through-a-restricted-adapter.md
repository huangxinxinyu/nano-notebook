# Fetch public URLs through a restricted Adapter

Known public HTML and document URLs will be imported only through a least-privileged Fetcher Adapter rather than the browser-facing Control Plane or Agent. It accepts only HTTP(S), resolves and validates all IPv4 and IPv6 destinations against private, loopback, link-local, reserved, and metadata ranges, revalidates every bounded redirect, and enforces response size, decompression, content-type, and time limits without user cookies, authorization headers, database access, or durable credentials. Network policy will restrict production egress in depth. Successful content is snapshotted as an immutable Source input before normal processing; the Fetcher provides no search, recursive crawl, authenticated-page, or arbitrary protocol capability.
