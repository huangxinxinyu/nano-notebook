# Normalize multimodal Sources into one text Retrieval Channel

The first release will use Gemini to extract OCR, structured descriptions, and region-linked evidence from images, figures, rendered pages, and slides, then index that normalized text alongside document text, HTML, captions, and audio transcripts. It will not create a native visual-vector Collection or cross-channel fusion; original media and stable region coordinates remain available for Citations, and native multimodal retrieval can be added later without changing Source authority.
