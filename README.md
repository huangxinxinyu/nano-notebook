# Multica Delivery Framework

This repository contains the local-first Multica Delivery Framework V1. It is a
Codex-led workflow scaffold for requirements, planning, implementation, QA,
review, recovery, and lightweight memory capture.

## Trigger Contract

The framework is explicit-trigger only. Ordinary coding requests do not start
the workflow. Use it only when the user explicitly asks for the Multica
delivery workflow or invokes `$multica-delivery`.
